# 基本情報技術者試験 科目B 模擬試験

基本情報技術者試験（FE）の科目B（午後試験）を想定した模擬試験アプリです。擬似言語によるアルゴリズム・プログラミング問題をブラウザで練習できます。

## 機能

- **40問**の擬似言語問題（整列・探索・再帰・データ構造）
- **80分**のタイマー（1問あたり2分想定）
- 問題の進捗・旗マーク・正誤表示
- ローカル保存・復元（同一ブラウザ内）
- 採点後の解説表示
- ショートカットキー対応

## 使い方

### 1. 起動

ES モジュールを使用しているため、ローカルサーバーで起動してください。

```bash
npx serve .
```

または、VS Code の Live Server 等で `index.html` を開いてください。

ブラウザで `http://localhost:3000` を開きます。

### 2. 試験の流れ

1. 問題番号をクリックして問題を選択
2. 選択肢（ア〜エ）をクリックして解答
3. 「旗を付ける」で復習したい問題にマーク
4. 「保存」で進捗を保存（同一ブラウザ内のみ有効）
5. 「提出して採点」で採点結果と解説を表示
6. 「不正解へ移動」「旗の問題へ移動」で復習

### 3. ショートカットキー

| キー | 動作           |
| ---- | -------------- |
| 1    | 選択肢アを選択 |
| 2    | 選択肢イを選択 |
| 3    | 選択肢ウを選択 |
| 4    | 選択肢エを選択 |
| ←    | 前の問題へ     |
| →    | 次の問題へ     |

## プロジェクト構成

```
fe-b-cbt-mock/
├── index.html      # HTML と CSS
├── data/
│   ├── code.js     # 擬似言語コード
│   └── questions.js# 問題データ
└── js/
    └── app.js      # アプリケーションロジック
```

## 出題テーマ

| テーマ                 | 問題数   |
| ---------------------- | -------- |
| 整列（選択ソート）     | 10問     |
| 整列（バブルソート）   | 9問      |
| 整列（比較）           | 1問      |
| 探索（線形探索）       | 5問      |
| 探索（二分探索）       | 5問      |
| 再帰                   | 5問      |
| データ構造（スタック） | 5問      |
| **合計**               | **40問** |

## 注意事項

- 提出前は解説は表示されません
- 提出後は各問題で正誤と解説が表示されます
- 保存・復元はこのブラウザ内のみ有効です
- 採点状態を固定すると回答の変更はできません
